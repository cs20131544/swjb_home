## ✨ 주요 특징

- **단일 HTML 파일** — 별도 빌드 도구 없이 바로 실행 가능
- **히어로 배경 영상** — `videoplayback.mp4` 자동재생, 음소거, 0~33초 구간 반복 루프
- **코딩 터미널 애니메이션** — About 섹션에 Python 코드가 실시간으로 타이핑되는 인터랙티브 요소
- **`< IT소프트웨어과 >` 로고** — HTML 태그 스타일의 학과 정체성을 담은 네비게이션 로고
- **반응형 디자인** — 모바일·태블릿·데스크탑 모두 대응
- **스크롤 애니메이션** — 각 섹션 진입 시 fade-in 효과

## 📷 이미지 교체 방법

`images/` 폴더 안의 파일명을 동일하게 유지한 채 새 사진으로 교체하면 자동 반영됩니다.

히어로 배경 영상 교체 시 `videoplayback.mp4` 파일을 같은 이름으로 덮어쓰면 됩니다.
